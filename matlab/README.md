# 柱状图（bar）
只介绍纵向的

## 2d柱状图

file:`bar_2d.m`

* 'FaceColor'：柱状图里面的颜色
* 'EdgeColor'：柱状图边框的颜色
* 'LineWidth'：柱状图边框粗细

![bar2d](img/bar_2d.jpg)
